Go-EMD

- Go Language: EMbarrassingly Distributed

Go-EMD allows you to write projects that will take control of a cluster of machines perform tasks that are too big or too resource consuming for one machine to do. This is written in GoLang so it is runnable on all machines that have GoLang installed.

What's GoLang (Go)?

GoLang is a programming language owned by Google built for great resource management and utilization of cores allowing for great concurrency programming. -- See more on the GoLang website to learn the language.

Why use Go-EMD?

If you have a problem set that is resource intensive or just a task that needs to be performed with many machines being synchronized with each other, then Go-EMD is a solution. It provides a framework for building a distribution easily, a GUI to monitor and view the health of the ditribution, and provides a quick and easy way to perform agile development.
